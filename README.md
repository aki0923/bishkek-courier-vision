#  Bishkek Courier Vision

> Last 100 meters navigation for courier in Bishkek

##  Problem
Couriers waste 20% of their time finding the exact entrance, gate codes, and doorbells inside residential complexes.

##  Solution
Crowdsourced mobile app where couriers share entrance photos, intercom codes, and tips with each other.

##  Tech Stack

### Backend
- PHP 8.1
- MySQL 8.0
- Composer

### Mobile
- Flutter 3.16+
- Dart

### AI Service
- Python 3.9
- Flask
- OpenAI Vision API

### DevOps
