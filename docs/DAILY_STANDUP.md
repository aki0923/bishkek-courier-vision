# Daily Standup Schedule

## Время и место
- **Время**: Каждый день в 09:00
- **Длительность**: Максимум 15 минут
- **Формат**: Все стоят (чтобы не затягивать)

## Структура

### Каждый участник (по кругу, 3 минуты на человека):

**1. Что сделано вчера?**
- Конкретные коммиты/задачи
- Завершенные фичи

**2. Что буду делать сегодня?**
- Конкретные задачи из Trello
- Ожидаемые результаты

**3. Есть ли блокеры?**
- Технические проблемы
- Нужна помощь команды
- Ждете чего-то от других

### Правила
-  НЕ решаем проблемы на standup
-  НЕ обсуждаем код детально
-  Только статус и блокеры
-  Детальные обсуждения - после standup отдельно

## Расписание Week

### Понедельник 09:00
- Обзор недели
- Распределение задач
- Настройка окружения

### Вторник 09:00
- Прогресс по API
- Frontend integration plan
- AI service status

### Среда 09:00
- Integration checkpoints
- Testing начинается
- Презентация outline

### Четверг 09:00
- Bug bash результаты
- Final polish tasks
- Presentation rehearsal

### Пятница 09:00
- Final checklist
- Demo run-through
- Подготовка к pitch
