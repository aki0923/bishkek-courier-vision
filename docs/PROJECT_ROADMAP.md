# Bishkek Courier Vision - Project Roadmap

- **Project Manager**:  - Координация, Trello, документация
- **Backend Developer**:  - PHP API, база данных
- **Frontend Developer**:  - Flutter мобильное приложение

## MVP Features (Must Have)
1.  Авторизация курьера
2.  Список адресов

- Работающий MVP к четвергу вечером

- Docker контейнер запускается
- Trello board показывает прогресс
