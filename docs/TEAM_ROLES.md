## Communication Channels

### Daily Standup Format (15 минут)
Каждый отвечает на 3 вопроса:
1. Что я сделал вчера?
2. Что буду делать сегодня?
3. Есть ли блокеры?


### Code Review Process
1. Create feature branch
2. Make changes
3. Push and create Pull Request


### Conflict Resolution
Если есть технические споры:
1. Обсудить в команде (5 минут)
2. Если нет согласия - PM принимает решение
3. Приоритет: работающий MVP > идеальный код
